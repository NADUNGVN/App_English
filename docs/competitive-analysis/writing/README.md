# Writing Analysis

Khối này dùng để phân tích sản phẩm luyện viết tiếng Anh.

## Scope

- Sentence writing
- Paragraph writing
- IELTS/TOEIC writing support
- Grammar and clarity feedback
- Rewrite suggestions
- Idea structure
- Draft history

## Questions

- Đối thủ sửa câu, sửa đoạn, hay hướng dẫn lập ý?
- Feedback có quá dài không?
- Người học có biết sửa gì trước không?
- Có giữ lịch sử draft để thấy tiến bộ không?
- QuackUp nên làm writing thành vòng lặp ngắn ra sao?
