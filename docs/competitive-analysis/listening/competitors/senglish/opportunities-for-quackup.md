# Opportunities For QuackUp

Competitor: S English  
Module analyzed: Dictation  
Source: https://senglish.net/vi/dictation  
Date: 2026-04-29

## 1. Make Listening Sessions Shorter

S English appears to emphasize real video-based practice. QuackUp can reduce the first-session load by starting with one short segment and one clear task.

QuackUp direction:

- 3-5 minute listening sessions.
- One current sentence in focus.
- One primary action: listen, type, check.

## 2. Explain The Listening Error

Dictation often tells learners what text is wrong, but not why they missed it. QuackUp can classify the error.

Possible error types:

- Missed weak sound.
- Missed linking.
- Confused similar sounds.
- Understood meaning but missed exact wording.
- Did not know the phrase.

## 3. Save Phrases With Context

Instead of saving isolated words, QuackUp should save:

- Phrase.
- Source sentence.
- Audio/video timestamp.
- Learner's wrong attempt.
- Correct sentence.

## 4. Bridge Dictation To Shadowing

After a learner fixes a hard sentence, QuackUp should prompt:

- Listen one more time.
- Speak after the speaker.
- Compare rhythm or completion.

This connects listening and speaking without making them separate islands.

## 5. Keep AI Secondary

AI support is useful, but the first screen should not become an AI chat product. QuackUp should expose AI only after the learner has made an attempt or asks for help.

Good default:

- Main screen: audio, input, feedback.
- Secondary panel: explanation, phrase meaning, examples.

## Candidate MVP Requirements

- Segment player with repeat current line.
- Dictation input per segment.
- Immediate answer check.
- Phrase save with context.
- Next-step CTA to shadow the same line.
- Basic session summary: completed lines, accuracy, saved phrases.
