# Scoring Rubric

Dùng thang điểm 1-5 để so sánh các đối thủ trong cùng skill block.

| Score | Meaning |
|---|---|
| 1 | Yếu hoặc chưa thấy rõ |
| 2 | Có nhưng rời rạc |
| 3 | Đủ dùng |
| 4 | Tốt, đáng học nguyên lý |
| 5 | Rất mạnh, cần phân tích sâu |

## Criteria

| Criterion | Câu hỏi đánh giá |
|---|---|
| Clarity | Người học có hiểu ngay mình cần làm gì không? |
| Practice Loop | Vòng lặp luyện tập có ngắn, rõ, lặp lại được không? |
| Feedback Quality | Feedback có chỉ đúng lỗi và bước sửa tiếp theo không? |
| Content Fit | Nội dung có phù hợp trình độ/mục tiêu không? |
| Progress Signal | Người học có thấy tiến bộ không? |
| Retention Hook | Có lý do quay lại hằng ngày không? |
| Product Focus | Module có tập trung hay bị loãng bởi quá nhiều tính năng? |
| QuackUp Opportunity | Có cơ hội làm rõ hơn, nhanh hơn, hoặc khác biệt hơn không? |
