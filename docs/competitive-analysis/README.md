# Competitive Analysis

Thư mục này dùng để phân tích website đối thủ trước khi quyết định tính năng, luồng học, nội dung và UI cho QuackUp.

Mục tiêu không phải sao chép đối thủ. Mục tiêu là hiểu:

- Đối thủ giải quyết vấn đề học tiếng Anh nào.
- Người học đi qua luồng nào để hoàn thành một phiên học.
- Điểm nào làm trải nghiệm tốt hoặc kém.
- QuackUp nên học nguyên lý nào và biến thành giải pháp riêng.

## Skill Blocks

```text
competitive-analysis/
├── reading/
├── listening/
├── speaking/
└── writing/
```

Mỗi skill block có thể có nhiều đối thủ. Mỗi đối thủ nên có một thư mục riêng dưới `competitors/`.

## Quy Ước

- Dùng tiếng Việt cho nhận định sản phẩm.
- Dùng tên thư mục tiếng Anh, chữ thường, có gạch ngang nếu cần.
- Mỗi phân tích phải ghi rõ URL nguồn và ngày phân tích.
- Tách rõ `observed`, `inferred`, và `opportunity` để tránh nhầm dữ kiện với ý tưởng.
- Không copy nguyên văn nội dung marketing hoặc UI copy dài từ đối thủ.
