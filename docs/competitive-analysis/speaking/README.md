# Speaking Analysis

Khối này dùng để phân tích sản phẩm luyện nói tiếng Anh.

## Scope

- Shadowing
- Pronunciation feedback
- Speaking rhythm
- Response practice
- Recording and playback
- Fluency scoring
- Conversation or roleplay

## Questions

- Đối thủ giúp người học nói lại, phản xạ, hay hội thoại?
- Feedback tập trung vào phát âm, nhịp, ngữ điệu hay nội dung?
- Người học có nghe lại giọng mình không?
- Có tách câu dài thành đoạn ngắn dễ luyện không?
- QuackUp nên nối speaking với listening như thế nào?
