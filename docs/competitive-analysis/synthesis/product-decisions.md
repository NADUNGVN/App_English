# Product Decisions From Competitive Analysis

This file records decisions that should influence QuackUp implementation.

## Decided

- Keep the four analysis blocks: reading, listening, speaking, writing.
- Treat Dictation as part of the listening block.
- Treat Shadowing as a bridge between listening and speaking, not only a standalone speaking feature.

## Candidate Decisions

- Listening MVP should focus on short segment loops before broad content browsing.
- Vocabulary saved from listening should include sentence/audio context.
- AI explanation should be secondary and appear after a learner attempt.

## Needs Evidence

- Whether learners prefer full-sentence dictation or blank-fill dictation.
- Whether YouTube-based content creates too much choice friction.
- Whether browser extension is relevant before QuackUp has strong core sessions.
