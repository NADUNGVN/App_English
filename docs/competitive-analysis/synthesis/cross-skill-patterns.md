# Cross-Skill Patterns

Use this file to collect patterns that repeat across reading, listening, speaking, and writing competitors.

## Current Patterns

| Pattern | Seen in | Product meaning |
|---|---|---|
| Short active loop | Listening / Dictation | Learners need a concrete task, not only content consumption |
| Context capture | Listening / Vocabulary | Saved items should keep source context |
| Skill bridge | Listening to Speaking | A completed listening line can become a shadowing line |
| Helper layer | AI / Dictionary | Support tools should help after effort, not replace practice |

## Watchlist

- Do strong competitors separate skill pages or blend skills into one session?
- Which feedback is immediate, and which feedback is delayed?
- Which products make progress visible without overwhelming the learner?
