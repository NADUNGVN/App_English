**Frontend:** React + HTML + CSS + JavaScript

**Backend:** Python + Lambda + SQS + Linux

**Database/Data Storage:** DynamoDB + RDS + S3 + Kafka

**Trong đó:**

**Ở Frontend:**

* **HTML** : tạo khung giao diện
* **CSS** : tạo giao diện, màu sắc, bố cục
* **JavaScript** : xử lý hành vi và tương tác
* **React** : thư viện/framework hỗ trợ xây dựng UI

**Ở Backend:**

* **Python** : xử lý logic nghiệp vụ
* **Lambda** : chạy xử lý backend theo chức năng
* **SQS** : hàng đợi message để xử lý tác vụ
* **Linux** : môi trường/hệ thống chạy backend

**Ở Database/Data Storage:**

* **DynamoDB** : lưu dữ liệu NoSQL
* **RDS** : cơ sở dữ liệu quan hệ
* **S3** : lưu file, ảnh, tài nguyên
* **Kafka** : xử lý luồng dữ liệu/message

**Tóm tắt chức năng:**

* **Frontend** : phần giao diện để người dùng nhìn thấy và thao tác
* **Backend** : xử lý nghiệp vụ, đăng nhập, xử lý dữ liệu
* **Database** : lưu tài khoản, lưu trữ dữ liệu và tài nguyên hệ thống
